## Quick Start

Get started with Skeletor UI in minutes:

```bash
# Install dependencies
npm install @radix-ui/react-* class-variance-authority clsx tailwind-merge

# Copy components to your project
# Browse components at /docs/components
```

## Architecture

Skeletor UI is built on these foundational technologies:

- **React 18** - Latest React features including RSC support
- **Radix UI** - Unstyled, accessible components  
- **Tailwind CSS** - Utility-first CSS framework
- **CVA** - Class variance authority for component variants
- **Next.js** - Full-stack React framework

## What's Included

### Core Components
- **Layout**: Cards, Separators, Containers
- **Navigation**: Breadcrumbs, Navigation Menu, Pagination  
- **Forms**: Input, Select, Checkbox, Radio, Switch
- **Feedback**: Alert, Dialog, Tooltip, Progress

### Advanced Components  
- **Data Display**: Table, Chart, Empty States
- **Interactive**: Command, Resizable, Collapsible
- **Specialized**: Button Groups, Input Groups, Field components
